    Net-cat

    NetCat is a versatile project featuring a Server-Client Architecture. It can operate in server mode, listening for connections on a specified port, or in client mode, connecting to a specified port to transmit data. As a command-line utility, NetCat (nc) reads and writes data across network connections using TCP or UDP, capable of opening TCP connections, sending UDP packets, and listening on various ports. This implementation leverages Go-routines and mutexes for efficient concurrent processing and synchronization, enhancing its performance and reliability.
